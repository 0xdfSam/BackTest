# IB Quant Project

This project demonstrates a basic quantitative trading setup with Interactive Brokers integration and backtesting capabilities.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Configure your IB account details in `config.py`
3. Ensure IB Trading Workstation is running and configured to accept API connections

## Usage

1. Run backtesting: `python main.py backtest`
2. Run live trading: `python main.py live`
